# 环境准备

## 1 大模型 API 配置

国内外适合学习使用的大模型服务平台推荐。以学习 RAG 为目标，梳理推荐排序如下。

- [硅基流动](https://siliconflow.cn/)

   国内 AI 开发者几乎都知道。特点：注册送额度、模型丰富、价格低。

- [阿里百炼](https://dashscope.aliyun.com?utm_source=chatgpt.com)
  
  有一定的免费额度，支持模型丰富。

- [AIHubmix](https://aihubmix.com/?aff=anNj&lang=zh-CN)
    有一定的免费额度，支持模型丰富。嵌入、重排序模型没有免费额度，支持支付宝充值。

- [google ai studio](https://aistudio.google.com?utm_source=chatgpt.com)
   
   目前最慷慨的免费额度之一，免费额度长期存在，不是试用期。支持的模型比较有限。

- [NVIDIA NIM](https://build.nvidia.com/)
    
  有免费额度，这个平台目前不大熟悉
